# Scoop Bucket for gf

[gf](https://github.com/josinSbazin/gf) is a CLI for GitFlic (Russian GitHub alternative).

## Install

```powershell
scoop bucket add gf https://github.com/josinSbazin/scoop-gf
scoop install gf
```

## Update

```powershell
scoop update gf
```

## Uninstall

```powershell
scoop uninstall gf
scoop bucket rm gf
```
